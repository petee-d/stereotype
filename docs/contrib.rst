Optional dependencies
=====================

These features intentionally stay outside of the core stereotype implementation as they may introduce dependencies.

Schematics adapter
------------------

Use this only if already depending on `Schematics <https://github.com/schematics/schematics>`_.
If migrating from Schematics, be sure to check out the
`migration guide <https://github.com/petee-d/stereotype/blob/master/stereotype/contrib/README.md#schematics>`_.
